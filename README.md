# agrosynapse-agrinho
O AgroSynapse Pro é um site interativo criado para o Concurso Agrinho 2026. A plataforma conecta o campo e a cidade, trazendo uma calculadora de impacto ambiental (CO₂ e água), dados sobre o solo das regiões do Brasil, curiosidades educativas e recursos de acessibilidade como modo escuro e leitor de tela.
